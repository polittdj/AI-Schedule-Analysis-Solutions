# Skill Library

This directory holds the Claude Code skill library for the Schedule Forensics
Local Tool. Each subdirectory is a single skill with a `SKILL.md` manifest;
skill bodies are populated by subsequent sessions.

## Skills

- `nasa-schedule-management/` — NASA-published schedule management practices and conventions.
- `nasa-program-project-governance/` — NASA program/project governance framework, lifecycle phases, and milestones.
- `dcma-14-point-assessment/` — DCMA 14-point schedule health assessment metrics and thresholds.
- `driving-slack-and-paths/` — Critical path, driving path, and float/slack forensic analysis patterns.
- `forensic-manipulation-patterns/` — Forensic signatures of schedule manipulation (status overrides, constraint abuse, duration compression).
- `mpp-parsing-com-automation/` — Microsoft Project .mpp parsing via MPXJ / JPype1 and Windows COM-automation reference.
- `acumen-reference/` — Deltek Acumen Fuse metrics, thresholds, and terminology for cross-reference.
- `cui-compliance-constraints/` — Controlled Unclassified Information handling rules and dual-mode AI routing constraints.

## Status

All eight skills currently contain **placeholder `SKILL.md`** files only. Content
will be populated in subsequent sessions. Do not treat the placeholder
descriptions as authoritative.
