What runs when you click Analyze?

  • MPP parser reads each file locally via MPXJ/JPype.
  • Auto-detect order reads each file's status date and sorts chronologically before analysis.
  • Forensic engine computes CPM, DCMA 14-point, earned value, float, and (for 2+ files) comparator, manipulation detection, and delay root cause.
  • Trend analysis runs when you upload 3+ files — builds time-series for float erosion, SPI drift, and baseline resets.
  • AI narrative is not generated automatically — open the Analysis tab and click "Generate AI Analysis" when you're ready.

The forensic engine handles ~90% of the work deterministically. AI only writes the narrative from pre-computed metrics.
